DATA STRUCTURE REFERENCE
================================================================================

Each EOD Report document contains:

{
  animalsFed: "24/24",              ← String, format: "X/Y"
  medsGiven: 8,                     ← Number
  volunteersCount: 5,               ← Number
  status: "submitted",              ← String
  notes: "...",                     ← String (observations)
  issues: [                         ← Array of issue objects
    {
      id: "1",                      ← String (unique in report)
      animal: "Animal Name",        ← String
      issue: "Description...",      ← String
      priority: "high|med|low"      ← String
    }
  ],
  createdAt: Timestamp              ← Date/Timestamp
}
