BEST PRACTICES
════════════════════════════════════════════════════════════════════════════════

DO:
  ✅ Use CSS variables for all values
  ✅ Follow the spacing scale
  ✅ Use semantic colors for states
  ✅ Add animations to interactions
  ✅ Test in both dark and light modes
  ✅ Check focus states and contrast

DON'T:
  ❌ Hardcode colors like #FF7A45
  ❌ Use arbitrary spacing like padding: 23px
  ❌ Create custom shadows
  ❌ Skip focus states
  ❌ Override animations without reason
  ❌ Mix naming conventions
